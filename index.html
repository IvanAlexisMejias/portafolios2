Sensor de Temperatura a Tiempo Real

Este proyecto utiliza tecnologías modernas para monitorear la temperatura en tiempo real.

Esta es una aplicación web en la cual el usuario se registra con un correo, luego debe logearse donde entra un a un menu principal
Donde Vera un Grafico a tiempo real de la temperatura que se actualiza cada 5 segundos, El usuario establece un umbral que se marca en el grafico
Cada vez que la temperatura sobrepase el Umbral establecido, el usuario sera notificado en su Email (con el cual se registro)

Sistema de Tarificación Telefónica

Aplicación móvil diseñada para calcular tarifas telefónicas de forma eficiente.

Desarrollado con Kotlin y Android Studio, este sistema utiliza Firebase para almacenamiento y autenticación.
En esta aplicacion para Android, El usuario debe registrarse, y ceder permisos de informacion de llamadas,cada vez que el usuario reciba
una llamada, La aplicacion lo registrara en Fire base y devolvera a la app la informacion completa de la llamada (numero, fecha, duracion)
ademas de una tarificacion ficcticia en CLP's de la llamada
  • Kotlin
  • Java
  • Android Studio
  • Firebase
